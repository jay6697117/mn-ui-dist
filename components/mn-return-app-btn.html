# mn-return-app-btn app返回悬浮框

组件名：mn-return-app-btn 代码块： mn-return-app-btn 关联组件：mn-icons，mn-fab

# 安装方式

本组件符合easycom (opens new window)规范，HBuilderX 2.5.5起，只需将本组件导入项目，在页面template中即可直接使用，无需在页面中import和注册components。

如需通过npm方式使用uni-ui组件，另见文档：https://ext.dcloud.net.cn/plugin?id=55 (opens new window)

# 基本用法

从 APP 跳转到小程序的场景值为 scene=1069，可根据他来进行判断展示和隐藏。

在 template 中使用组件

<mn-return-app-btn></mn-return-app-btn>

# API

# fab Props

属性名 类型 默认值 说明
propAppName String - '返回' 返回名称
vertical String - bottom 垂直对齐方式。bottom:下对齐，top：上对齐
distance Number - 220 距离顶部或底部的初始化距离，单位px
move Boolean - true 是否可以上下拖动位置
show Boolean - false 是否展示

# Events

事件名 说明 返回值
@returnApp 点击返回按钮 -
@close 点击关闭按钮 -

# 组件示例

<mn-return-app-btn :show="true" propAppName="爱奇艺" vertical="top" :move="true" distance="200"></mn-return-app-btn>
