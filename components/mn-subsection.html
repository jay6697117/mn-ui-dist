# mn-subsection 分段器

组件名：mn-subsection 代码块： mn-subsection

# 安装方式

通过npm方式使用mn-mall-ui组件 本组件符合easycom (opens new window)规范，HBuilderX 2.5.5起，只需将本组件导入项目，并配置 easycom 规则后，在页面template中即可直接使用，无需在页面中import和注册components。

# 基本用法

在 template 中使用组件

    <mn-subsection active-color="red" mode="subsection" :list="subsectionlist" :current="subsectionCurrent"/>

# API

# 属性

属性名 类型 默认值 可选值 说明
list Array [] - 数据列表
current String/Number 0 - 当前选择项
active-color String #303133 - 激活状态颜色
inactiveColor String #606266 - 未激活状态颜色
mode String button button/subsection 模式
fontSize String/Number 28rpx - 字体大小单位rpx
animation boolean true true/false 过度动画
height String/Number 70rpx - 高度
bold boolean true true/fasle 激活状态字体加粗
bgColor String #eeeeef - 背景色，只有mode==button时有效
buttonColor String #ffffff - 滑块背景颜色，只有mode==button时有效
vibrateShort boolean false true/false 切换时震动反馈

# 事件

change

# 组件示例

	<template>
		<mn-subsection active-color="red" mode="subsection" :list="subsectionlist" :current="subsectionCurrent"/>
	</template>

	<script>
		export default {
			data() {
				return {
					subsectionlist: [
                        {
                            name: '待发货'
                        },
                        {
                            name: '待付款'
                        },
                        {
                            name: '待评价'
                        }
                    ],
                    subsectionCurrent: 0,
				}
			},
		}
	</script>
