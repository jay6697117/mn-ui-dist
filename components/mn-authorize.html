# mn-authorize 授权登录

组件名：mn-authorize 代码块： mn-authorize

登录授权组件

# 安装方式

通过npm方式使用mn-mall-ui组件 本组件符合easycom (opens new window)规范，HBuilderX 2.5.5起，只需将本组件导入项目，并配置 easycom 规则后，在页面template中即可直接使用，无需在页面中import和注册components。

# 基本用法

在 template 中使用组件

<mn-authorize :isHidden="isHidden" @onHidden="isHidden = true"></mn-authorize>

# API

# authorize Props

属性名 类型 默认值 说明
isHidden Boolean false 是否隐藏授权组件
needAuthPhone Boolean true 是否需要微信手机授权
isAutoShowLayer Boolean true 在未登录状态下是否自动显示弹框
isCloseGoIndex Boolean true 点击关闭授权是否跳页
indexPageUrl String '/pages/index/index' 点击关闭授权按钮，跳转页面地址；isCloseGoIndex 为true 时生效
isCustomStyle Boolean false 是否自定义样式

# authorize vuex state

属性名 类型 默认值 说明
logoUrl String '' 授权logo

# authorize vuex action

事件名 事件说明 传递参数 返回参数
@getLogoAction 加载组件时，没有logoUrl 会自动触发 -
@checkAuthStatusAction 监测当前是否已经登录，加载组件时自动触发 - {status, userInfo} status： 0 未授权，1微信用户授权，2手机号授权； userInfo：用户信息，status 不为 0 时返回
@bindUserInfoAction 点击绑定微信授权用户信息 wx.getUserProfile 信息 - {} 需返回用户userInfo
@bindPhoneAction 点击 手机号授权 触发事件 getPhoneNumber 信息 - {} 返回用户userInfo

# authorize Events

事件名 事件说明 返回参数
@onHidden 点击关闭授权触发 -
@onAuthFun 授权完成触发 -
