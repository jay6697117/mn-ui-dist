# mn-custom-modal 自定义对话框

组件名：mn-custom-modalmn-custom-modal 代码块： mn-custom-modal 关联组件：mn-icons

# 安装方式

本组件符合easycom (opens new window)规范，HBuilderX 2.5.5起，只需将本组件导入项目，在页面template中即可直接使用，无需在页面中import和注册components。

如需通过npm方式使用mn-mall-ui组件，另见文档：https://ext.dcloud.net.cn/plugin?id=55 (opens new window)

# 基本用法

在 template 中使用组件

<mn-custom-modal :active="customModalActive" @close="customModalActive = false" :title="'modal-title'" :isTopClose="true" :isMaskClose="true" :modalStyle="'background-color:#fff;'">
    <template v-slot:content>
        <view class="content" style="margin-top: 20px">测试内容</view>
    </template>
</mn-custom-modal>

# API

# mn-custom-modal Props

属性名 类型 默认值 说明
active boolean - false 是否显示弹框
isTopClose String - true 是否显示右上角关闭按钮
title String - '' 弹框title
modalStyle String - '' 弹框自定义样式css
titleStyle Boolean - true title 自定义样式
isMaskClose Boolean - true 点击背景是否关闭

# mn-custom-modal Events

事件名 说明 返回值
@close 关闭回调事件 -
