# mn-rule-fab 规则弹框组件

组件名：mn-rule-fab 代码块： mn-rule-fab 关联组件：mn-custom-modal mn-fab

# 安装方式

本组件符合easycom (opens new window)规范，HBuilderX 2.5.5起，只需将本组件导入项目，在页面template中即可直接使用，无需在页面中import和注册components。

如需通过npm方式使用mn-mall-ui组件，另见文档：https://ext.dcloud.net.cn/plugin?id=55 (opens new window)

# 基本用法

在 template 中使用组件

<mn-rule-fab ruleFabTitle="使用规则" modalTitle="规则说明" :contentHtml="contentHtml" :titleStyle="'color:red'"></mn-rule-fab>

# API

# mn-rule-fab Props

属性名 类型 默认值 说明
ruleFabTitle String - '规则说明' 规则浮框文案
ruleFabTitleStyle String - '' 规则浮框自定义样式
ruleFabHorizontal String - 'right' fab水平对齐方式。left:左对齐，right：右对齐
ruleFabVertical String - 'top' fab垂直对齐方式。bottom:下对齐，top：上对齐
ruleFabCustom Boolean - false 是否自定义fab，使用slot传值
distance Number - 80 fab距离顶部或底部的初始化距离，单位px
modalTitle String - '' 规则弹框标题文案
modalShow Boolean - false 控制 modal 展示与隐藏
contentHtml String - '' 规则弹框内容，支持html
titleStyle String - '' 规则弹框标题自定义样式
modalStyle String - '' 规则弹框自定义样式
confirmStyle String - '' 规则弹框确认按钮自定义样式
confirmTxt String - '我知道了' 规则弹框确认按钮文案
modalCustom Boolean - false 用于自定义modal，使用slot

# mn-rule-fab Events

事件名 说明 返回值
@modalShow 规则弹框展示变更回调 - true 表示弹框展示，false 表示弹框隐藏
