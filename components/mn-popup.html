# mn-popup 弹出层

组件名：mn-popup 代码块： mn-popup

自定义弹出层

# 安装方式

通过npm方式使用uni-ui组件 本组件符合easycom (opens new window)规范，HBuilderX 2.5.5起，只需将本组件导入项目，并配置 easycom 规则后，在页面template中即可直接使用，无需在页面中import和注册components。

# 基本用法

在 template 中使用组件

<mn-popup mode="bottom" :showClose="true" title="demo" :show="true">
	<view>
		自定义显示内容
	</view>
</mn-popup>

# 属性

属性名 类型 默认值 可选值 说明
mode String bottom top
bottom
left
right
center		 弹出方向
show boolean false true / false 是否显示
safeArea boolean true true / false iphonex适配底部安全区
maskCloseAble boolean true true / false 是否点击遮罩层关闭，如果是，show需要加.sync修饰 即 show.sync="xxx"
showClose boolean true true / false 是否显示关闭按钮
title string 空 - 弹出成标题
duration String/Number 100 - 动画时间
borderRadius String/Number 0 - 弹出层圆角大小
zIndex [String, Number] 9999 - 弹出层的z-index值
customStyle Object 空 - 弹出层内容的自定样式

# slot

default 弹出层具体视图插槽
