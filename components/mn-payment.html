# mn-payment 支付方式弹窗

组件名：mn-payment 代码块： mn-payment

# 安装方式

通过npm方式使用uni-ui组件 本组件符合easycom (opens new window)规范，HBuilderX 2.5.5起，只需将本组件导入项目，并配置 easycom 规则后，在页面template中即可直接使用，无需在页面中import和注册components。

# 基本用法

在 template 中使用组件

<mn-payment :paymentList="[]" show="true" />

# 属性

属性名 类型 默认值 可选值 说明
show boolean false true / false 是否显示
paymentList Array 微信支付 - 支付方式列表
showPayMoney boolean false true / false 是否要显示支付的金额

# paymentList数据结构

属性名 类型 说明
name String 支付名称
icon String icon图标,内置的icon图标，如果不能满足可以传递一个图片网络图片地址
iconColor String icon的样式，只有是内置的icon才有效
desc String 支付方式的描述
money String number

eg:

	[
		{
			name: "微信支付",
			icon: "wechat",
			iconColor:'#52AC38',
			desc: '微信快捷支付',
			money: 100,
		},
		{
			name: "余额",
			icon: "https://mn-duoshanghu.oss-cn-beijing.aliyuncs.com/xndj/5d289202107301116465556.jpg",
			desc: '微信快捷支付',
		}
	]
