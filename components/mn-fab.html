# mn-fab 悬浮框

组件名：mn-fab 代码块： mn-fab 关联组件：mn-icons

# 安装方式

本组件符合easycom (opens new window)规范，HBuilderX 2.5.5起，只需将本组件导入项目，在页面template中即可直接使用，无需在页面中import和注册components。

如需通过npm方式使用mn-mall-ui组件，另见文档：https://ext.dcloud.net.cn/plugin?id=55 (opens new window)

# 基本用法

在 template 中使用组件

<mn-fab></mn-fab>

# API

# fab Props

属性名 类型 默认值 说明
custom boolean - false 是否自定义内容
horizontal String - right 水平对齐方式。left:左对齐，right：右对齐
vertical String - bottom 垂直对齐方式。bottom:下对齐，top：上对齐
distance Number - 220 距离顶部或底部的初始化距离，单位px
move Boolean - true 是否可以上下拖动位置
pattern Object - { color: '#fff', backgroundColor: 'rgba(179, 150, 55, .6)', } 颜色配置
content Array - [ { url: '/pages/index/index', openType: 'switchTab', iconfont: 'shouye1', iconImg: '' }, { url: '/pages/order_addcart/order_addcart', openType: 'switchTab', iconfont: 'gouwuche', iconImg: '' }, { url: '/pages/user/user', openType: 'switchTab', iconfont: 'yonghu1', iconImg: '' } ] 展开菜单内容配置项

pattern配置项：

参数 类型 默认值 说明
color String #ffffff 文字默认颜色
backgroundColor String rgba(179, 150, 55, .6) 背景色

content配置项：

参数 类型 说明
url String 跳转路径
openType String 跳转方式
iconfont String 图标字体类型
iconImg Boolean 如没有图标字体，可以使用图片地址

# Icons Events

事件名 说明 返回值
@clickFabTrigger 点击fab 按钮事件 - 返回是否展开Boolean值
@clickFabItem 点击 fab 菜单 - 返回当前点击菜单数据
